# zhuomian
### 简介

一个可以将任意文件夹设置为桌面文件夹的小软件

### 用途

1. 桌面文件太多，又不能直接删除，需要进行备份后清理
2. 桌面太杂，临时给领导或者客户演示项目，需要更整洁的桌面
3. 录制视频时，临时需要干净的桌面，录制完后又要恢复原貌
